## 介绍
使用<a href="https://cn.vuejs.org/" target="_blank">vuepress</a>搭建的静态网站

## 开发
```
yarn install
yarn start
```

## 部署
使用<a href="https://travis-ci.org/" target="_blank">Travis CI</a>进行持续集成，提交源码后，会自动生成并部署静态网站
